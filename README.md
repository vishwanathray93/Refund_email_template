# Refund Email Template – Venue Marketplace

Professional and responsive refund email templates for Venue Marketplace bookings.  
This project helps businesses send clean and user-friendly refund confirmation emails to customers.

---

## ✨ Features

- Responsive email template design
- Modern and professional UI
- Easy customization
- Dynamic booking details support
- Refund confirmation email layout
- Compatible with most email clients
